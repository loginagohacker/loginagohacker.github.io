Changelog

Beta, Alpha, and Released versions are the classifcation for development stage. Beta is stll under heavy development (and is slightly buggy), Alpha is still under development (less buggy, more supports), and Released is the version that is released to the public (will have bug fixes every so often, new supports.)

New classifications: R means the released version of FlipAI, F is the final versions of the website (meaning it is finished, just small changes and bug fixes are periodically installed).

For example, the current version, as of 4/16/2021, is Version RF-4.0.26.49. The number 4 is published version of FlipAI, 0 is FlipAI bug fixes in released versions, 26 is published version of website, 49 is the number of bug fixes in the website.
